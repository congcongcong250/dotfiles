## Main usages

### Keep following layout for keyboard

[ Ctrl ] + [ Opt ] + [ Cmd ] + [     Space     ] + [ fn ] + ...

### Arrow keys as (key on the right of Space ) + i/k/j/l 

fn + i -> Up

fn + k -> Down

fn + j -> Left

fn + l -> Right
